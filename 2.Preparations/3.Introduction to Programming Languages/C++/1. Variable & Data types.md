### What is a variable?

A **variable** is:
> A named place in memory that stores a value.

Example:

`int age = 16;`
- `int` → type    
- `age` → name    
- `16` → value    

---

## Common C++ data types 

### 1️⃣ `int` – whole numbers

`int score = 100; int lives = 3;`

- No decimals    
- Usually 4 bytes    

---

### 2️⃣ `double` – decimal numbers

`double pi = 3.14; double temperature = 36.5;`

- Can store decimals    
- More precise than `float`    

---

### 3️⃣ `char` – single character

`char grade = 'A'; char symbol = '@';`

⚠️ Uses **single quotes** `' '`

---

### 4️⃣ `bool` – true or false

`bool isAlive = true; bool gameOver = false;`

Used heavily in conditions.

---

### 5️⃣ `std::string` – text

`#include <string>  std::string name = "Alex";`

- Stores words or sentences
    
- Uses **double quotes**
    

---

## Declaring vs assigning

### Declare only:

`int x;`

### Declare + assign:

`int x = 5;`

### Assign later:

`x = 10;`

---

## Rules for variable names

✅ Allowed:

`int playerScore; int _count;`

❌ Not allowed:

`int 2score;     // can't start with number int player-score; // no hyphens`

---

## Type matters (VERY important)

`int x = 5 / 2;       // result: 2 double y = 5 / 2;   // result: 2.0 double z = 5.0 / 2; // result: 2.5`

C++ follows the **types of the operands**, not the variable.
